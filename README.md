<h2 id="chinese">🇨🇳 中文说明</h2>

一款专为 WordPress Gutenberg 编辑器设计的轻量级插件。在上传图片至服务器之前，直接在浏览器本地将其压缩并转换为 WebP 格式，极大节省服务器带宽和存储空间，提升网站加载速度。

### ✨ 核心功能
* **本地高效压缩**：在浏览器端直接将 JPG/PNG 图片转换为 WebP 格式。
* **无缝集成 Gutenberg**：在编辑器侧边栏增加专属的“压缩上传”面板，完全不影响原生上传流程。
* **自定义压缩质量**：内置可视化滑块，默认 80% 质量，可自由调节（1-100）。
* **智能插入光标位置**：上传完成后，自动将 `core/image` 块精准插入到当前光标所在行的下方。
* **体积对比直观展示**：上传成功后，自动显示压缩前后的文件大小对比及节省百分比。
* **极简架构**：纯原生 JS 编写，无需 Webpack/Babel 等复杂构建工具。

### 🚀 安装与使用
1. 在 Releases 页面下载最新的 `.zip` 压缩包。
2. 登录 WordPress 后台，进入「插件」 -> 「安装插件」 -> 「上传插件」，选择 ZIP 文件并激活。
3. 打开任意文章或页面的 Gutenberg 编辑器。
4. 点击右上角「⋮」（更多工具），选择「压缩上传图片」打开侧边栏面板。
5. 调节质量滑块，选择图片，等待自动压缩上传并插入正文。

<h2 id="english">🇬🇧 English</h2>

A lightweight WordPress plugin designed for the Gutenberg editor. It compresses and converts images to WebP format locally in the browser before uploading them to your server, significantly saving bandwidth, storage space, and improving page load speeds.

### ✨ Key Features
* **Local Compression**: Converts JPG/PNG images to WebP directly in the browser.
* **Gutenberg Integration**: Adds a dedicated "Compress & Upload" panel to the editor sidebar.
* **Custom Quality Slider**: Built-in visual slider to adjust compression quality (default 80%, range 1-100).
* **Smart Cursor Insertion**: Automatically inserts the `core/image` block exactly below your current cursor position.
* **Before/After Size Comparison**: Displays a clear comparison card showing file size saved and percentage reduction after a successful upload.
* **Zero Dependencies**: Written in vanilla JS utilizing native WordPress components, no Webpack/Babel required.

### 🚀 Installation & Usage
1. Download the latest `.zip` file from the Releases page.
2. Go to your WordPress Dashboard -> Plugins -> Add New -> Upload Plugin, select the ZIP, and activate it.
3. Open any post or page in the Gutenberg editor.
4. Click the "⋮" (More tools & options) in the top right corner and select "Compress & Upload Image" to open the sidebar.
5. Adjust the quality slider, select your image, and watch it automatically compress, upload, and insert into your content.
